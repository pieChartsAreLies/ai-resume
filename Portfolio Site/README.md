# Portfolio Site

React SPA with tab navigation: Work, Resume, AI Approach, Contact.

## Dev

```bash
npm install
npm run dev
```

## Key Components

| Component | Description |
|-----------|-------------|
| `Work.tsx` | Project grid with category toggle, tag filtering, featured section |
| `WorkDetailPanel.tsx` | STAR-format detail overlays with screenshots and GitHub links |
| `GearSiftShowcase.tsx` | Lazy-loaded full-page showcase for GearSift project |
| `AIApproach.tsx` | "How I Work with AI" philosophy section |
| `Resume.tsx` | Experience timeline (4 roles) |

## Theme

Warm palette: `#FAF7F2` (cream text), `#D4A853` (gold accent), `#4A4440` (card bg), `#2A2622` (dark bg). Font: Montserrat.
